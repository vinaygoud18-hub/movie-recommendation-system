A content-based movie recommendation system that suggests movies similar to the one you pick — based on genres, keywords, cast, director and plot overview. Built as part of my Data Science journey at MRCET.

---

## 📌 What It Does

- 🎯 Pick any movie from 4,800+ titles
- ⚡ Get 10 similar movies instantly
- 🖼️ View movie posters via OMDb API
- 📊 See cosine similarity scores for each result

---

## 🧠 How It Works

```
Input Movie
    │
    ▼
Extract → genres + keywords + cast + director + overview
    │
    ▼
CountVectorizer → Bag of Words vector (5000 features)
    │
    ▼
Cosine Similarity → 4806 × 4806 matrix
    │
    ▼
Top 10 most similar movies
```

**Why Cosine Similarity?**  
It measures the angle between two vectors rather than distance — so a short and long overview about the same topic still score as similar, which makes it ideal for text data.

## 🚀 Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/vinaygoud18-hub/movie-recommendation-system.git
cd movie-recommendation-system

# 2. Install dependencies
pip install -r requirements.txt

# 3. Build the model (run once — takes ~30 seconds)
python setup.py

# 4. Launch the app
streamlit run app.py
```

Opens at `http://localhost:8501`

> **Movie Posters:** Get a free OMDb API key at [omdbapi.com/apikey.aspx](http://www.omdbapi.com/apikey.aspx) and paste it in the sidebar. Free tier gives 1,000 requests/day.
